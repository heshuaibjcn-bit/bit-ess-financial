# 投资建议

## 模板规范

### 开头模板

```
本章节旨在综合以上分析，给出明确的投资建议和行动建议。我们基于[分析框架]，并综合[评估结果]得出以下结论。
```

### 内容结构

#### 1. 综合评分

**要求：**
- 多维度评分（财务、政策、风险、市场）
- 加权平均总分
- 评分等级：优秀（≥90分）、良好（80-89分）、一般（70-79分）、较差（<70分）

**格式：**
```
综合评分：
• 财务维度：[分数]/100
• 政策维度：[分数]/100
• 风险维度：[分数]/100
• 市场维度：[分数]/100
• 加权总分：[分数]/100
• 评级：[优秀/良好/一般/较差]
```

#### 2. 投资建议

**要求：**
- 明确结论：推荐 / 谨慎推荐 / 不推荐
- 建议理由：综合财务、政策、风险、市场的分析
- 核心论据：支持建议的关键数据

**格式：**
```
投资建议：[推荐/谨慎推荐/不推荐]

建议理由：
基于以上分析，本项目[优势描述]。具体而言：
- 财务表现：[财务评价]
- 政策环境：[政策评价]
- 风险水平：[风险评价]
- 市场前景：[市场评价]

核心论据：
- IRR [X]% [高于/低于]行业平均水平
- NPV [金额] 万元 [表明价值创造/价值损失]
- 政策支持 [稳定/不稳定]，预计补贴收入 [金额] 万元/年
- 主要风险 [可控/不可控]
```

#### 3. 行动建议

**要求：**
- 立即行动项（如适用）
- 后续调研项（如适用）
- 条件件建议（如适用）

**格式：**
```
行动建议：
• 立即行动：
  - [行动1]：[描述]
  - [行动2]：[描述]
• 后续调研：
  - [调研1]：[描述]
  - [调研2]：[描述]
• 条件件：
  - [条件1]：[描述]
  - [条件2]：[描述]
```

### 结尾模板

```
本投资建议基于当前可获得的信息做出。实际投资决策应考虑更详细的尽职调查和市场调研。
```

## 输出格式

```typescript
interface InvestmentRecommendation {
  overallScore: {
    financial: number;        // 0-100
    policy: number;          // 0-100
    risk: number;            // 0-100
    market: number;          // 0-100
    weightedScore: number;   // 0-100
    rating: '优秀' | '良好' | '一般' | '较差';
  };
  recommendation: {
    conclusion: '推荐' | '谨慎推荐' | '不推荐';
    reasoning: string;        // 多段式理由
    keyEvidence: string[];   // 核心论据
  };
  actionItems: {
    immediateActions: Array<{
      action: string;
      description: string;
    }>;
    followUpResearch: Array<{
      research: string;
      description: string;
    }>;
    conditions: Array<{
      condition: string;
      description: string;
    }>;
  };
  disclaimer: string;
}
```
