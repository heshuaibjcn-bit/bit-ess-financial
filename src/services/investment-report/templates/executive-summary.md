# 执行摘要

## 模板规范

### 一段概述（150-200 字）

**要求：**
- 简明项目基本情况（业主、地点、规模）
- 说明投资规模（投资金额、系统容量）
- 总结核心结论（推荐/谨慎推荐/不推荐）

**示例：**
> 本项目为[业主名称]位于[地点]的[系统容量]工商业储能系统，总投资[投资金额]万元。基于全面的财务、政策和风险分析，项目[内部收益率IRR]为[X]%，净现值NPV为[金额]万元，投资回收期为[X]年，[投资建议]。

### 3-5 个关键要点

**要求：**
- 要点 1：核心财务指标（IRR、NPV、回收期）
- 要点 2：关键政策支持（补贴机制、电价政策）
- 要点 3：主要风险因素（技术、市场、政策）
- 要点 4：项目优势或亮点
- 要点 5：（可选）其他关键信息

**格式：**
```
• 财务表现：IRR [X]%，NPV [金额]万元，回收期 [X] 年
• 政策支持：享受[具体政策]，预计补贴收入 [金额] 万元/年
• 主要风险：[风险类型]风险，[缓解建议]
• 项目亮点：[核心优势，如地理位置好、业主实力强等]
```

### 明确的投资建议

**要求：**
- 明确结论：推荐 / 谨慎推荐 / 不推荐
- 一句话说明理由

**示例：**
> **投资建议：推荐**
> 理由：项目财务表现优秀（IRR 12.3%，NPV 230 万元），政策支持稳定，主要风险可控，建议立即投资。

## 输出格式

```typescript
interface ExecutiveSummary {
  overview: string;          // 一段概述（150-200 字）
  keyPoints: string[];        // 3-5 个关键要点
  recommendation: {          // 投资建议
    conclusion: '推荐' | '谨慎推荐' | '不推荐';
    reason: string;           // 一句话理由
  };
}
```
